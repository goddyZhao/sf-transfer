sf-transfer

Installation

npm install -g sf-transfer

Usage

sf -i input.xml -o ~/workspace

More Options

Options:

-h, --help             output usage information
-V, --version          output the version number
-i, --input [input]    Specify the sf configuration xml
-o, --output [output]  Specify the output
